# Учетная система CRM Demo

Фронтенд-демо учетной CRM, целиком работающее на мок-данных. Проект нужен для презентации интерфейса, сценариев и данных: вход по демо-ролям, общий реестр проектов, карточки проектов, контрагенты, объекты, КП-конструктор, каталог, оплаты, документы, задачи и аналитика.

## Что важно

- Демка полностью фронтендовая.
- Все CRM-данные лежат в `src/crm/mockData.ts`.
- Типы сущностей лежат в `src/crm/types.ts`.
- Локальное состояние демки лежит в `src/crm/demoStore.ts` и сохраняется в `localStorage`.
- Роли входа лежат в `src/crm/auth.ts`.
- КП-конструктор сохранен как модуль внутри демки.

## Где менять мок-данные

Для презентации обычно достаточно править:

- `src/crm/mockData.ts` - проекты, контрагенты, объекты, оплаты, документы, товары и поставщики.
- `src/crm/auth.ts` - демо-роли и пользователи входа.
- `src/data/demoData.ts` - старые данные КП-конструктора.

`src/crm/demoStore.ts` оставлен как единственная точка замены источника данных. Когда понадобится настоящий внешний источник, его можно подключать там, не переписывая UI-компоненты.

## Запуск

```powershell
npm install
npm run dev
```

Открывать:

```text
http://127.0.0.1:5173
```

Если порт занят, Vite покажет другой порт в терминале.
